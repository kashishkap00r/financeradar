River Explorations

4 prototypes reimagining FinanceRadar as a unified content river

1. The Ledger

Every item = one fixed-height row. IBM Plex Mono, 3-color palette, 28px rows. Like git log --oneline for news.

Monospace · Grid · Minimal

2. The Broadsheet

FT-inspired multi-column newspaper. AI rank drives visual hierarchy — bigger headline = more important. Salmon background.

Serif · Newspaper · Warm

3. The Tide Chart

Horizontal timeline. Time flows left-to-right, items stack as compact chips within each hour column.

Timeline · Chips · Clinical

4. The Folio

Museum exhibition grid. WSW debate clusters are top-level zones; individual items flow in a dense grid below.
