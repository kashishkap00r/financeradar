Editorial Bento as the New Home

A modular landing surface that pulls one high-quality slice from each section and uses clear View More exits into existing tabs.

Color System: Existing FinanceRadar Tokens

Home Intent
One glance should answer: what matters right now?

The home page is not a replacement for existing tabs. It is a highly scannable command surface combining News, Telegram, Reports, Papers, YouTube, and Twitter into curated preview blocks with strong hierarchy.

Preview Depth, Not Feed Exhaustion

Interaction Layer

Each block keeps one primary View More action and optional bookmark action. Avoid deep controls on Home to preserve speed.

AI Picks Strip (Optional)

Add an optional AI strip above the grid that surfaces one pick per section. Keep this hideable per user preference.

Visual Hierarchy

Lead module gets 1.5x height. Supporting modules stay compact. Typography and spacing do most of the work, not heavy decoration.

Performance Budget

Render Home from existing data arrays. Avoid fetching new payloads on first paint. Lazy load thumbnails after top fold.

0-200msTop bar and tabs visible
200-450msHome modules fade in with stagger
On tapView More routes to target tab + restores scroll
